## 1-erece: Ж = C (J emes)

Tubjolge Qyrğyz Ж dajyma **C c** /dʒ/. **J j** tamğasy **Й** /j/ üçün ğana.

| Bulaq | Latyn   | Qaçan                     | Misaldar                                 |
| ----- | ------- | ------------------------- | ---------------------------------------- |
| Ж     | **C c** | Tubjolge Qyrğyz           | жок→coq, жер→cer, жаңы→cañy, жабуу→cabuu |
| Ж     | **Ž ž** | Qaryz sözdör, özdük attar | журнал→žurnal, Брежнев→Brežnev           |
| Й     | **J j** | Dajyma (carym ündüü)      | бай→baj, ой→oj, кайда→qajda              |

### Köp taralğan tubjolge Ж sözdörü

- жок → **coq** (coq/eçteme)
- жокко чыгаруу → **coqqo** çyğaruu (öçürüü)
- жаңы → **cañy** (cañy)
- жана → **cana** (cana)
- жер → **cer** (cer/orun)
- жеке → **ceke** (ceke)
- жай → **caj** (orun/caj)
- жабуу → **cabuu** (cabuu)
- жабык → **cabyq** (cabyq)
- жардам → **cardam** (cardam)
- жарык → **caryq** (caryq)
- жаш → **caş** (caş/caş)
- жазуу → **cazuu** (cazuu/cazuu)
- жаныбар → **canybar** (canybar)
- жөнөтүү → **cönötüü** (cönötüü)
- жүктөө → **cüktöö** (cüktöö)
- же → **ce** (ce)

### Ž misaldary (qaryz sözdör ğana)

- журнал → žurnal
- Жарчысы журналы → Carçysy žurnaly
- жюри → žüri
- монтаж → montaž
- Брежнев → Brežnev

## 2-erece: K/Q cana G/Ğ ündüü ujğunduğu

Kirill К cana Г ar biri qurşağan ündüülörgö caraşa eki latyn tamğağa bölünöt.

**Artqy ündükter (a, o, u, y):** К → **Q**, Г → **Ğ**
**Aldyñqy ündükter (e, ö, ü, i):** К → **K**, Г → **G**

| Kirill | Latyn (artqy) | Latyn (aldyñqy) |
| ------ | ------------- | --------------- |
| к      | Q q           | K k             |
| г      | Ğ ğ           | G g             |

### Misaldar

Artqy ündüü sözdör:

- карыя → qaryja
- кагаз → qağaz
- артка → artqa
- кымыз → qymyz
- кооз → qooz
- аргымак → arğymaq
- байкалган → bajqalğan

Aldyñqy ündüü sözdör:

- көз → köz
- биргелешкен → birgeleşken
- үзгүлтүк → üzgültük

### Müçö ujğunduğu

| Müçö (Kir) | Artqy | Aldyñqy |
| ---------- | ----- | ------- |
| -ган/-ген  | -ğan  | -gen    |
| -гон/-гөн  | -ğon  | -gön    |
| -ка/-ке    | -qa   | -ke     |
| -ко/-кө    | -qo   | -kö     |
| -кы/-ки    | -qy   | -ki     |
