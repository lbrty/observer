## Kural 1: Ж = C (J değil)

Yerli Kırgızca Ж her zaman **C c** /dʒ/ olur. **J j** harfi yalnızca **Й** /j/ içindir.

| Kaynak | Latin | Ne zaman | Örnekler |
| ------ | ----- | -------- | -------- |
| Ж | **C c** | Yerli Kırgızca | жок→coq, жер→cer, жаңы→cañy, жабуу→cabuu |
| Ж | **Ž ž** | Alıntı kelimeler, özel isimler | журнал→žurnal, Брежнев→Brežnev |
| Й | **J j** | Her zaman (yarı ünlü) | бай→baj, ой→oj, кайда→qajda |

### Yaygın yerli Ж kelimeleri

- жок → **coq** (hayır/hiç)
- жокко чыгаруу → **coqqo** çyğaruu (silmek)
- жаңы → **cañy** (yeni)
- жана → **cana** (ve)
- жер → **cer** (toprak/yer)
- жеке → **ceke** (kişisel)
- жай → **caj** (yer/yaz)
- жабуу → **cabuu** (kapatmak)
- жабык → **cabyq** (kapalı)
- жардам → **cardam** (yardım)
- жарык → **caryq** (ışık)
- жаш → **caş** (yaş/genç)
- жазуу → **cazuu** (yazı/kayıt)
- жаныбар → **canybar** (hayvan)
- жөнөтүү → **cönötüü** (yönlendirme)
- жүктөө → **cüktöö** (yükleme)
- же → **ce** (veya)

### Ž örnekleri (yalnızca alıntı kelimeler)

- журнал → žurnal
- Жарчысы журналы → Carçysy žurnaly
- жюри → žüri
- монтаж → montaž
- Брежнев → Brežnev

## Kural 2: K/Q ve G/Ğ Ünlü Uyumu

Kiril К ve Г harfleri, ünlü ortamına bağlı olarak iki Latin harfine ayrılır.

**Kalın ünlüler (a, o, u, y):** К → **Q**, Г → **Ğ**
**İnce ünlüler (e, ö, ü, i):** К → **K**, Г → **G**

| Kiril | Latin (kalın) | Latin (ince) |
| -------- | ------------ | ------------- |
| к        | Q q          | K k           |
| г        | Ğ ğ          | G g           |

### Örnekler

Kalın ünlülü kelimeler:

- карыя → qaryja
- кагаз → qağaz
- артка → artqa
- кымыз → qymyz
- кооз → qooz
- аргымак → arğymaq
- байкалган → bajqalğan

İnce ünlülü kelimeler:

- көз → köz
- биргелешкен → birgeleşken
- үзгүлтүк → üzgültük

### Ek uyumu

| Ek (Kir)     | Kalın | İnce  |
| ------------ | ----- | ----- |
| -ган/-ген    | -ğan  | -gen  |
| -гон/-гөн    | -ğon  | -gön  |
| -ка/-ке      | -qa   | -ke   |
| -ко/-кө      | -qo   | -kö   |
| -кы/-ки      | -qy   | -ki   |

## Kural 4: Lehçe Harfleri

Lehçe metinlerinde bulunur (güney/Oş lehçesi, Afgan Kırgız halk edebiyatı).

| Harf   | Ses            | Örnekler                                   |
| ------ | -------------- | ------------------------------------------ |
| Ä ä    | açık ön /æ/    | asäl (= bal), gäp (= kep), mäkä (= cügörü) |
| W w    | dudak /w/      | yalnızca lehçe                             |

Kaynaklar: Murza Gaparov "Kyştakça", Sulajman Qajypov'un Afgan Kırgız halk edebiyatı çalışmaları, C. Muqambaev'in lehçe sözlüğü.
