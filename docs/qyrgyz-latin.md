## Rule 1: K/Q and G/Ğ Vowel Harmony

Cyrillic К and Г each split into two Latin letters depending on the vowel environment.

**Back vowels (a, o, u, y):** К → **Q**, Г → **Ğ**
**Front vowels (e, ö, ü, i):** К → **K**, Г → **G**

| Cyrillic | Latin (back) | Latin (front) |
| -------- | ------------ | ------------- |
| к        | Q q          | K k           |
| г        | Ğ ğ          | G g           |

### Examples

Back vowel words:

- карыя → qaryja
- кагаз → qağaz
- артка → artqa
- кымыз → qymyz
- кооз → qooz
- аргымак → arğymaq
- байкалган → bajqalğan

Front vowel words:

- көз → köz
- биргелешкен → birgeleşken
- үзгүлтүк → üzgültük

## Rule 3: Ж Handling

Cyrillic Ж has two Latin representations depending on origin.

| Context                          | Latin        | Example                       |
| -------------------------------- | ------------ | ----------------------------- |
| Native Kyrgyz words              | **C c** /dʒ/ | (standard alphabet letter #3) |
| Loanwords, proper names (soft Ж) | **Ž ž** /ʒ/  | журнал → žurnal               |

### Ž examples

- журнал → žurnal
- Жарчысы журналы → Carçysy žurnaly
- Жириновский → Žyrinovskij
- донжуан → donžuan
- Брежнев → Brežnev
- жандармерия → žandarmerija
- Женя → Ženä
- жюри → žüri
- монтаж → montaž

Note: Native Kyrgyz Ж uses C (e.g. жайлоо → cailoo), while foreign/borrowed Ж uses Ž.

## Rule 4: Dialect Letters

Found in dialectal texts (southern/Osh dialect, Afghan Kyrgyz folklore).

| Letter | Sound          | Examples                                   |
| ------ | -------------- | ------------------------------------------ |
| Ä ä    | open front /æ/ | asäl (= bal), gäp (= kep), mäkä (= cügörü) |
| W w    | labial /w/     | dialectal only                             |

Sources: Murza Gaparov "Kyştakça", Sulajman Qajypov's Afghan Kyrgyz folklore works, J. Muqambaev's dialect dictionary.
